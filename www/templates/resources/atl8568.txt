<div class="article-content-box">
<h3 class="subject">13岁少年和他父亲的命运之锤</h3>
<div class="intro">
<p>
亲友散去，唢呐声止，堂前屋后都恢复了宁静。屋内，只有那把铁锤可以见证和诉说罗积春曾有的失望、绝望和希望，是如何燃起，又是如何幻灭的。</p>
</div>
<div class="info">
<span class="author">
作者：本刊记者 韦星 发自湖南省衡南县</span>                <span class="author">
来源：南风窗</span>                <span class="author">
日期：2019-01-26</span>                                                            </div>
<div class="content">
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">冬季昼短，下午6点刚过，衡南的山村就被灰暗迅速笼罩。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">手抓一把铁锤，从工地摇晃出来时，罗积春感到体力难支。饥肠辘辘的他，骑上电动车，冒着冷风，朝三塘镇西南方向驶去。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">这天，恰逢圣诞节。途经三塘闹市时，各门店都在释放劲爆的揽客音乐：2019年就要到来了。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">在街边一家门店，罗积春停下来，买了副对联。对联就放在电动车的后备箱里，和在工地作业的铁锤一块。之后，一路颠簸回家。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">第二天，路过他家门口的村民发现，门上多了副对联：开福门心想事成，居宝地吉祥如意。横批：迎春接福。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">罗积春和他的妻子没有等来新年的幸福，等来了一场杀身之祸。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">2018年12月31日傍晚，他们13岁的儿子，抡起铁锤，砸向了自己的父母。</span></div>
<div>
 </div>
<div>
<strong><strong><strong><strong>　　</strong></strong></strong><span style="font-size:16px;">锤杀父母</span></strong></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">罗积春生于1967年，是湖南省衡南县三塘镇学塘村湾塘组的村民。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">湾塘组位于三塘镇西南方向的一个小山丘上，离镇上15公里。2019年的头几天，这个小村落迎来了北京、重庆、上海和广州等地的记者，这是建村以来从未有过的场景。大家直奔罗积春的家，探究“锤杀父母”的惨案如何发生。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">罗积春的家在村道边上，是一栋2层小楼，朝路边的墙面都贴上了瓷砖，但进入屋内，这栋6年前就建好的楼房，至今依旧是毛坯，没有任何装饰。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">屋内的电线，借助一些钉子缠绕和裸挂在墙上。除一台电冰箱和一部废弃了的厚厚的电视机外，屋内不再有其他家电。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">几乎空无一物，但这个男人依旧努力保持家庭该有的温暖。里屋靠墙的位置，堆满了给妻儿过冬烤火的煤球。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">如今，煤球还在，烤火的人却没有了。煤球周边，是一滩滩凝固了的黑褐色血迹，地上还有长条的拖行血迹。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">靠窗的位置还有一滩更大的血迹，血迹沿着墙面洒了下来，于地面形成了一汪汪的血迹。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">这是惨案现场留下的。2018年12月31日下午6时40分，13岁的罗飞先用铁锤锤杀母亲，尔后将母亲拖到里屋。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">父亲发现异常的动静后，披着睡衣从隔壁房间走过来，突然也被躲在门后的罗飞给锤杀了。“他父亲死的时候，连睡衣的扣子都没扣好。”村民罗家华说。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">锤杀父母后，罗飞搜走了父亲的1万多元，骑着电动车溜了。在村口300米处，他将电动车扔下斜坡，赶到10公里开外三塘镇上一家叫黄金岭的网吧上网。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">有媒体说罗飞锤杀父母后，在黄金岭网吧待2个多小时玩游戏。但学塘村党支部书记费益生说，只待19分钟，主要告诉同学，“我犯事了，需要出一趟远门。”</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">之后，他用父亲的身份证，买了前往云南的火车票。40多小时后的1月2日下午，罗飞在云南大理落网。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">获此消息，罗积春的大哥罗积生向亲友宣布：“落网了！”</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">说着，放声大哭。诸多亲友和罗积生一样，不知如何形容此刻的心情。“锤杀我弟弟和弟媳的，毕竟也是我侄子！”罗积生说，他身上毕竟流淌着罗家的血脉。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">“对他恨又无奈！”罗积生的大儿媳贺姐一声长叹，泪水在眼眶不停打转，几度欲言又止。</span></div>
<div>
 </div>
<div>
<strong><strong><strong><strong>　　</strong></strong></strong><span style="font-size:16px;">突然变坏？</span></strong></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">案发后，罗积春的大女儿跑去向大伯通报，有先天性智力障碍的她不能清晰说起事发的经过，只在大伯面前不断举起拳头，砸向自己脑袋。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">罗积生跟侄女跑到她家发现：弟弟罗积春靠窗斜倒在墙上，脑袋耷拉，鲜血汩汩外流。进入里屋，弟媳双腿跪地，脑袋垂倒在一堆黑色煤球上。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">罗积生向侄女询问案发经过，得到的始终是一句：“钱，游戏。”</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">村民猜测，罗飞是要钱玩游戏未果，这才锤杀了父母。因为，2018年12月28日学校放假时，罗飞没有回家，而是待在网吧玩游戏。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">12月29日，他在微信朋友群说，“我在黄金岭（网吧）包夜”，他还说“我逃学3天在网吧熬夜，谁有我这么牛逼？谁有我这么厉害吗？”</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">“只要给他吃，给他饮料，他可以一直玩下去，玩不腻的。”同学小陈这样描述罗飞对枪战游戏的迷恋程度。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">罗飞八年级的语文课本封面上，歪扭写着“时（使）命召唤”四字，这是一款枪战游戏的名字。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">12月30日晚，罗积春和班主任将罗飞从镇上找回，但他并不甘心。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">一个星期前，罗积春让罗飞预交下学期的1000元学费，他没交，且花掉了800元。乱花钱玩游戏，家庭矛盾由此诱发。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">从网络世界强行被抽离，带回到现实世界，罗飞也积抑着一肚子的不满。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">罗飞率先将不满的情绪发泄到母亲身上。父亲察觉到异常动静后，从其他房间走过来时，就被躲在门后的罗飞，举起2斤重的铁锤，挥向了父亲脆弱的脑壳。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">13岁的罗飞为何突然变坏，变狠？亲人、邻居、老师和同学，一遍遍向记者回忆他们曾熟悉，但此刻非常陌生的罗飞的点点滴滴。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">记者从这屋到那屋，从一楼到二楼，一遍遍翻找罗飞留下的任何蛛丝马迹，比如察看他课本，察看他作文，盯住墙壁上的一些画画，希望从中获得罗飞的杀人密码，但终无所获。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">因为无论是偶尔的抽烟耍酷、逃课上网、为朋友打架，或大方请同学吃饭，这都是青春期孩子的常见行为，很难把他和一个杀人犯—特别是和一个杀害自己父母的人联系在一起。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">“平时没看出什么坏苗头，否则，我们肯定会帮我叔教育他，甚至我叔也从没和我们提起孩子难教的话题。”贺姐说。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">罗家华和罗积春是小学同桌。平时路过罗积春家门口，印象中，他看到的罗飞比较内向，平时就拿个手机玩。此外，没看出异常。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">在老师布置的作文中，罗飞对父母的描述也都很好。描述母亲时，他用“很亲切，很贤惠”来形容妈妈。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">描述父亲时，罗飞也不吝赞美：“很朴实，说话和蔼，总先问我吃得好不好，其次才问我的成绩。别人对他很尊重，每次我和爸爸去买菜，认识的人都会亲切地和他打招呼，我为此感到骄傲。”</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">作文中对父母的好印象，更让人们对罗飞的突然出狠手，感到吃惊。当然，作文不代表罗飞的真实想法，毕竟有虚构的成分。比如，他就曾虚构母亲的家在云南，还说外婆的身体很矫健。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">其实，他母亲是湖南省郴州市安仁县人，外婆也过世多年。</span></div>
<div>
<span style="font-size:14px;">不过，作文中介绍邻居对母亲的态度时，罗飞强调的“邻居”是“小区邻居”。事实上，截至案发，他一直居住在湾塘这村落，没所谓“小区”。</span></div>
<div>
 </div>
<div>
<strong><strong><strong><strong>　　</strong></strong></strong><span style="font-size:16px;">沉重的家庭</span></strong></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">罗飞的母亲谭金花属先天性智力障碍，表达能力弱，不懂和别人交流。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">出事后，有记者问家属一个问题：罗积春当初为何娶选择这样一个妻子？毕竟，他健康、老实、勤快。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">其实，20世纪90年代以后，老实和勤快的农民已不再是女性择偶的重要参考标准。随着改革春风刮起，能言善辩的小伙子更易讨姑娘欢心。何况，依靠农耕改变命运的时代已远去，老实勤恳的男人在择偶市场上并不吃香。罗积春这样老实巴交的农民，除了一身的力气和1.65米的身材，他一无所有。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">时光来到1997年时，罗积春已是30岁的大龄青年。此时，昔日同桌罗家华已是一个7岁孩子的父亲。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">罗积春急了。这年，刚好有人给他介绍谭金花，他没拒绝，并火速完成传宗接代任务。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">1998年，罗积春的第一个孩子出生了，是个女儿，他很开心，但这种开心没持续多久。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">随着女儿长大，罗积春发现，女儿遗传了妻子的先天性智力障碍。和妻子一样，女儿的智识能力非常低，也缺乏清晰的表达能力，无法和别人正常交流。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">罗积春是家中顶梁柱，一旦垮了，妻女怎么办？他需要再生一个，但也担心：万一再生还遗传妻子的智识能力，非但无法救助这个家庭，还因此雪上加霜。犹豫很久，他不敢再生。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">一晃就过了8年，当日子于他在工地的不断挥锤和劳作中，终于有了改善后，他有了“再赌一把”的勇气。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">2005年6月，罗积春的第二个孩子—罗飞出生。这回，生了个男孩，而且看起来正常。罗家沉浸在欢声笑语中。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">彼时，衡阳城镇化加快，环衡阳市区的衡南县也迎来难得的机遇。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">衡南县下辖的三塘镇距离衡阳市政府，不过10公里。三塘的板块迅速崛起。目前，三塘镇已是个聚集超10万人口的大镇，商业繁华，拥有三塘工业园，更拥有大批地产商进驻和开发。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">房地产商的大举进驻和开发，提供大量工作机会。没文化，但有力气、扎实肯干的罗积春，终于迎来“不出远门也有改变人生的机会”。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">这些机会的抓取，只需他拥有一把铁锤。</span></div>
<div>
 </div>
<div>
<strong><strong><strong><strong>　　</strong></strong></strong><span style="font-size:16px;">命运之锤</span></strong></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">由于为人老实，干活卖力，做事勤恳，罗积春在工地很受欢迎。从泥水工做到装模工，即便在三塘，他一天收入也有300元左右。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">他那把花20元买来的铁锤，终成了改变他人生的希望之锤。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">随着罗飞一天天长大，看着他日渐聪颖，积抑在罗积春内心的不安，逐渐烟消云散。这意味着，依靠他的努力和罗飞的正常，家庭将一天天向好。想到这些，罗积春的干劲更大了。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">“每天凌晨四五点他就起床操持家务了，之后去三塘或衡阳市的工地打工，晚上八九点才回到家，”罗家华说，“春节前，罗积春经常干到腊月二十七八才停工。”</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">因老婆和女儿都患有智力障碍，指望不上，所以除了操持家务外，他还操心儿子的教育问题。在罗家华眼中，罗积春也不是只顾挣钱的那种，“他一有空，就会带儿子去逛街，也给他买好多好吃的东西。”</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">罗积春的二侄媳谢姐也向《南风窗》记者证实：“我叔对罗飞很好，平时冰箱里都给他存放排骨等好吃的东西。”</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">但家庭重压都压在罗积春一个人身上，他不可能挤出更多时间陪孩子。待在家的罗飞，尽管常年都有妈妈和姐姐陪伴，但由于她们的先天性智力障碍，他找不到可以倾诉和交流的对象。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">但靠一把铁锤，每天早出晚归，罗积春家的日子还是渐渐向好。6年前，他将房子建到两层，并将朝路面的墙壁都贴上瓷砖。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">2017年7月，罗飞从学塘小学毕业。同年9月，他进入大山学校就读初一。在大山学校，48人的班级里，他的成绩可以排到25-26名。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">在大山学校，罗飞没有异常的表现，“唯一被媒体放大的，就是说他为帮朋友争女友，花30元钱请高年级学生帮朋友打架。”该校一名教师告诉《南风窗》记者，其他都算正常。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">大山学校是所公办学校，收费较低，教育质量还可以，但罗积春望子成龙心切，希望给儿子最好的。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">2017年9月，三塘镇来了所“据说最好的学校”—华星学校。这所学校由衡南县一中和广东鹏诚教育集团合作办学，宣称是所“高端民办学校”。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">因办学时间短，是否高端目前还没到接受成果检验的时刻，但校舍无疑高端大气上档次。当然，学费也贵，这所学校一年收费1.5万元，是大山学校的10倍。为孩子的未来，一向节俭的罗积春舍得。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">罗积春曾花几千元给罗飞买学习机，后来发现，他拿去打游戏了。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">为了罗飞的教育，老实的罗积春也有过“出格”行为。入住新房后，他曾给家里添一台液晶电视，后来发现罗飞放学后天天守着电视看，“我叔一气之下，抡起铁锤砸烂了液晶电视。”谢姐说，他太焦心了。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">焦心的还有他女儿。3年前，罗积春的女儿曾有一次外嫁的机会，但没成功，女儿嫁出一个多月后，被她的娘家人退了回来。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">罗积春很烦恼，这意味着：和同龄人相比，罗飞今后将承担更多的赡养责任和压力。罗积春心疼儿子，心疼他生在这样的家庭里，他希望在力所能及的范围内，再拉儿子一把。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">2018年，罗积春到镇上给儿子买了套商品房。“这套在建的商品房，总价40多万元，我叔付了27万元，还剩10多万元。”贺姐说，这个家的开支，都是叔叔一锤子一锤子“抠”出来的。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">一切渐渐向好，但距离2019年不到6个小时，罗飞，这个13岁的少年，抡起了父亲谋生的锤子，砸向自己的双亲。罗飞毁掉了自己父母的生命，也毁掉了这个家，毁掉了自己的希望和未来。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">2019年1月1日，贺岁的对联，最终换成了白底黑字：天崩地裂血溅残阳云无光，风寒鹤泪断送双亲凤楼暗。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">1月4日上午8时50分，湾塘组上空，细雨纷飞。一阵阵哀怨的唢呐声中，16人一组，共32个老年人，他们抬着两部黑色的棺材，一前一后，摇晃上山，出殡了。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">不一会儿，亲友出殡归来，他们将罗积春家门口那副白底黑字的对联撕下，重新贴上那副金灿灿的“迎接春福”对联。屋前也打扫得干干净净的，似乎伤悲未曾来过。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">葬礼结束后，罗积春的女儿被政府接到当地福利院安置。</span></div>
<div>
<strong style="font-size:12.6316px;"><strong><strong><strong>　　</strong></strong></strong></strong><span style="font-size:14px;">亲友散去，唢呐声止，堂前屋后都恢复了宁静。屋内，只有那把铁锤见证了罗积春曾有的失望、绝望和希望，是如何燃起，又是如何幻灭。</span></div>
<div>
